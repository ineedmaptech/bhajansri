# ஸ்ரீ பஜன் சங்கீதம் 🕉️
### Tamil Devotional Song Database — PWA

A mobile-friendly Progressive Web App to store, search and manage Tamil bhajan songs with full metadata.

## Features
- 🎵 Store songs with Ragam, Thalam, Deity, Category, Composer, Date
- 🔍 Search and filter by any metadata
- ✏️ Add, Edit, Delete songs
- 💾 Offline storage via LocalStorage
- 📤 Export/Import JSON backup
- 📊 Export to CSV for Google Sheets
- 📱 Install on iPad/iPhone home screen

## How to Install on iPad
1. Open the app URL in Safari
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **Add** — done! App appears like a native app

## Files
- `index.html` — Main app
- `manifest.json` — PWA manifest
- `sw.js` — Service Worker for offline support
- `icon-192.png` / `icon-512.png` — App icons
